## INFO0940 - Operating Systems - Project 1

# How it works
When in the project directory, type in the terminal 
'''bash
make
'''
which will compile the codes. Then, use
'''bash
\[valgrind\] ./oshell
'''
Where 'valgrind' is used for debugging.
